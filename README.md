# ShiftQuickStart
A collection of .bat files

# start.bat

+ [ITH website]
+ [Papercut page]

After a 75 second delay it will open
+ [The rota]
+ [Article hub]
+ [ALT]
+ [gmail]
+ [mymaps]
+ [Library classmark tool]
+ [Traka]

It also opens notepad


# group policy.bat
Fetches and applies the latest group policy and logs out automatically
